Javascript

1. javascript与html代码的结合。

第一种方式：在<script>标签中直接写入js语句。例如：

<script type = "text/javascript">
    alert("hello");
</script>

第二种方式：将js语句保存为.js文件，然后在<script>标签中用src属性进行导入。例如：

<script type = "text/javascript" src = "demo.js"></script>

注意：如果<script>标签中使用了src属性，那么该标签中封装的js代码不会被执行。所以通常导入js文件都是使用单独的<script>来完成。

    自定义字符串功能：去除字符串两端的空格。

    function trim(str){
    var start, end;
    start = 0;
    end = str.length-1;

    while(start<=end && str.charAt(start)){
        start++;
    }
    while(start<=end && str.charAt(end)){
        end--;
    }
    return str.substring(start, end+1);
}

    prototype属性:

    String.prototype：String的原型。 原型：该对象所属的一个原有描述。 prototype可以获取到这个原型的对象。通过这个对象进行类功能的扩展。

    将上面创建的功能添加到字符串的对象的功能中。

    String.prototypr.len = 199; //给String的原型对象中添加一个属性。名为len，值为199.

    String.prototype.trim = trim; //将trim()方法添加到String对象的功能中。

    String.prototype.trim = function(){ var start, end; start = 0; end = this.length-1; while(start<=end && this.charAt(start)){ start++; } while(start<=end && this.charAt(end)){ end--; } return this.substring(start, end+1); } //或者通过匿名函数添加新的方法。

  10. js自定义对象。

    如果想要自定义对象，应该先对对象进行描述。js是基于对象而不是面向对象，不具备描述事物的能力。在js中，可以用函数来模拟面向对象中的描述。

    //用js来描述人。相当于构造器。
function Person(){
    alert("person run");
}

//通过描述进行对象的建立。
var p = new Person();
//动态给p对象添加属性，直接使用p.属性名即可。
p.name = "zhangsan";
p.age = 20;
//如果定义的p对象的属性赋值为一个函数，即是给p对象添加一个方法。
p.show = function(){
    alert("show:"+this.name+":"+this.age);
}
p.show();   //调用方法。


//另一种方法：
function Person(name, age){
    this.name = name;
    this.age = age;
    this.setName = function(name){
        this.name = name;
    }
    this.getName = function(){
        return this.name;
    }
}
var p = new Person("zhangsan", 20); //调用方法。
p.setName("Peter");
p.getName();
