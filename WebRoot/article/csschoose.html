CSS选择器

就是指定CSS要作用的标签，标签的名称就是选择器。 选择器共有三种：

  1. html名称选择器，使用的就是html的标签名。
  2. class选择器，其实使用的是标签中的class属性。
  3. id选择器，其实使用的是标签中的id属性。

每一个标签都定义了class属性和id属性。用于对标签进行标识。方便对标签进行操作。多个标签的class属性值可以相同，而id值必须唯一。因为该属性除了给css使用，还可以被js使用。id通常都是为了去标识页面中的一些特定区域。

<style type = "text/css">
    div{
        background-color:#09F;
        color:#FFF;
    }
    div.one{
        background-color:#FF3;
        color:#0C0;
    }
</style>

<div>这是一个div区域1</div>
<div class = "one">这是一个div区域2</div>

使用class属性来调用定义的属性。可以在<style>中定义多个样式。预定样式之后可以实现动态加载。

.oneplus{
    background-color:#FF0;
    color:#F00;
    }

这种方式定义以后可以在不同标签中使用该样式。

同样，可以利用id来进行选择。 在<style>中定义：

div#two{
    background-color:#FF0;
    color:#F00;
}

<div id = "oneplus">这是一个div区域2</div>

同理，也可以利用另一种定义方法。

#two{
    background-color:#FF0;
    color:#F00;
}

注意：优先级:

标签选择器<类选择器<id选择器<style属性

还有另外一种：选择器中的选择器。

在<style>标签中定义：

span a b{
    background-color:#FF0;
    color:#F00;
}

表示<span>标签中的<a>标签中的<b>标签。对此标签的样式设置。

还有还有另外一种：组合选择器。

在<style>标签中定义：

.oneplus,div b{
    background-color:#FF0;
    color:#F00;
}

表示.oneplus选择器和<div>标签中的<b>标签进行相同样式的定义。

还有还有还有另外一种：伪元素选择器。

伪元素： 超链接的状态：

1.未访问。
a:link{
    background-color:#06F;
    color:#FFF;
    text-decoration:none;
    font-size:20px;
}

2.鼠标悬停。
a:hover{
    background-color:#FFF;
    color:#F00;
    font-size:24px;
}

3.点击效果。
a.active{
    background-color:#000;
    color:#FFF;
    font-size:36px;
}

4.访问后效果。
a.visited{
    background-color:#FF9;
    color:#000;
    text-decoration:line-through
}
